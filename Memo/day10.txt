상속(inheritance)
	부모 클래스로부터 자식 클래스가 코드를 물려받는 것
	
	기존에 만들어놓은 클래스의 필드를
	새롭게 만들 클래스에서 자유롭게 사용하고 싶을 때
	활용하는 문법
public class 클래스명 extends 클래스명{

}

	상속을 받으면 자식클래스는 부모클래스의 멤버들을
	마음대로 사용할 수 있다
	자식에서 새롭게 필요한 멤버들은 새롭게 정의하면 된다.


오버라이딩(over riding)
	재정의
	부모클래스에 존재하는 메소드 이름과
	똑같은 이름으로 메소드를 다시 정의하는 것을 의미한다.

	부모클래스타입객체에서는 부모에서 정의된대로,
	자식클래스타입객체에서는 자식에서 재정의된대로 실행된다.
	다형성을 구현하는 요소 중 하나이다.

@Override
	어노테이션 주석
	해당 메소드가 오버라이딩 된 메소드임을 명시적으로 표현하고,
	부모 클래스에 메소드가 존재하지 않을 경우 오류를 발생시킴으로써
	개발의 실수를 줄일 수 있다.

super
	부모클래스를 의미하는 키워드
	super.메소드()를 통해 부모에 정의된 메소드를 호출할 수 있다.
	super()를 통해 부모생성자를 호출할 수 있다.

상속을 받은 자식 클래스의 생성자는
부모생성자를 먼저 호출한 후에 사용을 해야한다.

기본생성자의 경우 생략해도 컴파일러가 자동으로 추가해준다


public class A{
	public A(){}
}

public class B extends A{
	public B(){
		super();
	}
}

캐스팅(Casting)
	업캐스팅
		부모타입에 자식 값을 넣는 문법
	다운캐스팅
		자식타입에 부모 값을 넣는 문법

캐스팅의 목적
	여러 타입을 하나로 묶어 관리할 수 있다
